# Spire Tutorial Token Issuer

This service is for **TUTORIAL ONLY** which is meant to make the Spire node attestation smooth for learners.

## How to use

GET `/issue-token` and use the returned token in the spire-agent's [join_token plugin](https://github.com/spiffe/spire/blob/main/doc/plugin_agent_nodeattestor_jointoken.md).
