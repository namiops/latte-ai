# Intro

## access and accounts

We use the shared group [wcm-agora-observibility-admin] to access [grafana.com].
The login is shared via 1pass.

Only use this account as fall-back. Create your own administrator account using this
(or by asking your fellow team-mates).

## Grafana Cloud Support

### Ticketing system
  - Create tickets using the `Get Support` link on [grafana.com].
  - Put `wcm-agora-observibility-admin@woven-planet.global` in CC
  - Mention the environment you want support for
  - Tick the box to allow access for Grafana support if applicable

### Slack channel
  - `#wcm-agora-grafana-connect` on  (you need to be invited).
